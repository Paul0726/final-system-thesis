# Notification Message Templates para sa Respondents

## Message 1: Request para sa Additional Survey (Tagalog - Polite)

```
Mahal naming mga BSIT Graduate,

Kumusta po kayo? Umaasa kami na nasa mabuti kayong kalagayan.

Nais po naming ipaalam sa inyo na mayroon kaming karagdagang survey na kailangan naming makumpleto. Ang survey na ito ay bahagi ng aming patuloy na pagsisikap na mapabuti ang aming programa at matulungan ang mga susunod na batch ng mga estudyante.

Ang bagong survey ay may kasamang "System Evaluation" section na tutulong sa amin na mas maintindihan kung paano namin mapapabuti ang sistema. Ang inyong mga sagot ay lubhang mahalaga sa amin.

Paano po makumpleto ang survey:
1. Pumunta sa inyong Personal Dashboard gamit ang link sa ibaba
2. Mag-log in gamit ang inyong email address
3. Punan ang bagong "System Evaluation" section
4. I-submit ang inyong sagot

Ang survey ay maaari ninyong sagutan sa inyong Personal Dashboard. Kung mayroon kayong mga katanungan o kailangan ng tulong, huwag po kayong mag-atubiling makipag-ugnayan sa amin.

Maraming salamat po sa inyong oras at kooperasyon. Ang inyong kontribusyon ay malaking tulong sa pagpapabuti ng aming sistema.

Lubos na gumagalang,
BSIT Graduate Tracer System
Divine Word College of San Jose
```

## Paano gamitin:

1. Pumunta sa Admin Page
2. Click ang "Send Notifications" button
3. Piliin ang recipient (All Respondents, Employed, Unemployed, etc.)
4. Copy-paste ang message na gusto mo
5. Click "Send Notification"

**Note:** Ang system ay automatic na maglalagay ng instructions at link sa Personal Dashboard, kaya hindi mo na kailangan maglagay ng link sa message.
