# Paano Makakuha ng Gmail App Password

## Step-by-Step Instructions

### Option 1: Direct Link (Pinakamadali)

1. **Pumunta dito:** https://myaccount.google.com/apppasswords
2. Kung hindi ka pa naka-login, mag-login muna
3. Kung may lumabas na "Select app" at "Select device", piliin:
   - **Select app**: Mail
   - **Select device**: Other (Custom name)
   - Type: "BSIT Tracer System"
   - Click **Generate**
4. **Copy ang 16-character password** (walang spaces)

### Option 2: Through Security Settings

1. Pumunta sa: https://myaccount.google.com/security
2. Hanapin ang **"2-Step Verification"** section
3. Click ang **"2-Step Verification"** (yung may "On since..." na status)
4. Scroll down hanggang makita mo ang **"App passwords"** section
5. Click **"App passwords"**
6. Piliin:
   - **Select app**: Mail
   - **Select device**: Other (Custom name)
   - Type: "BSIT Tracer System"
   - Click **Generate**
7. **Copy ang 16-character password**

### Option 3: Kung Hindi Makita ang App Passwords

Kung hindi mo makita ang "App passwords" option:

1. **Check kung naka-enable ang 2-Step Verification:**
   - Dapat may "On" status
   - Kung hindi, i-enable muna

2. **Try i-refresh ang page** o mag-logout at login ulit

3. **Check kung naka-enable ang "Less secure app access":**
   - Pumunta sa: https://myaccount.google.com/lesssecureapps
   - I-enable kung available (pero mas better ang App Password)

4. **Alternative: Gumamit ng Gmail-specific password:**
   - Pumunta sa: https://myaccount.google.com/apppasswords
   - Kung wala pa rin, try mag-search sa Google Account settings: "app password"

## Important Notes

- **App Passwords ay available lang kapag naka-enable ang 2-Step Verification**
- Ang App Password ay 16 characters, may spaces (halimbawa: `abcd efgh ijkl mnop`)
- Kapag inilagay sa Railway, **i-remove ang spaces** (halimbawa: `abcdefghijklmnop`)

## Kung Hindi Pa Rin Makita

1. **Verify na naka-enable ang 2-Step Verification:**
   - Pumunta sa: https://myaccount.google.com/security
   - Dapat may "On" status sa 2-Step Verification

2. **Try mag-access via direct link:**
   - https://myaccount.google.com/apppasswords

3. **Kung work account ang Gmail:**
   - Baka naka-disable ng admin ang App Passwords
   - Kailangan i-enable ng admin

4. **Alternative Solution:**
   - Pwede mong gamitin ang regular password kung hindi available ang App Password
   - Pero mas secure ang App Password
